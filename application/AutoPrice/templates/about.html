Intro

When shopping for a new car most consumers are unaware of all the ancillary costs associated with owning a vehicle. These costs include:

  • Fuel
  • Maintenance and repair
  • Financing charges
  • Insurance
  • Depreciation

There are really only three things a new car buyer generally knows:

  1. What style of vehicle they want (car, truck, or SUV).
  2. How much they can affoard to pay a month.
  3. What the sticker price (MSRP) of the new car is.

AutoPrice attempts to solve this problem by using machine learning algorithms to predict the 5-year cost of ownership using only two features: the type of vehicle the user is interested in buying and either how much the user can afford a month or how much the car they want sells for.

How It Works

AutoPrice is a project developed for the 2017 Insight Data Science Remote Program. It uses a combination of web scraping tools and machine learning to predict the total cost of ownership given the style and price of a vehicle. The data come from both the Department of Energy's fuel economy website as well as True Cost to Own® data from the Edmunds.com webpage.

The Edmunds data is taken as the target value and with which the following assumptions are taken into considerations:

  • Ownership expenses are estimated for a 5-year period.
  • The vehicle will be financed to own and not leased.
  • A 10% down payment will be made.
  • The user has above average credit (700+).
  • The loan term is 60 months.

A linear regression algorithm utilzing python's SciKitLearn module is performed on the data and is used to predict that total cost of ownership. The linear regression algorithm was chosen over Support Vector Machine (SVM) and Random Forest algorithms because it gave the smallest mean absolute error while keeping the user input requirements to a minimum.
